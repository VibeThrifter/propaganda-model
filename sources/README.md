# Bronmateriaal

Plaats hier teksten en transcripts die later geanalyseerd worden om entiteiten en relaties te inferren.

## Mappen

- **transcripts/** — Uitgeschreven audio/video (interviews, podcasts, Kamerdebat)
- **articles/** — Nieuwsartikelen, columns, opiniestukken
- **interviews/** — Interview-transcripts en samenvattingen

## Naamgeving

Gebruik bij voorkeur: `YYYY-MM-DD_spreker_onderwerp.txt`

Bijvoorbeeld: `2024-03-15_baudet_mediakritiek.txt`
